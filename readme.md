
To run:

- clone electron-forge-app-webpack-d3
- `cd electron-forge-app-webpack-d3`
- `npm install && npm start`


Initial creation:

```
node: v12.14.1
npm: 6.13.4
electron-forge: 6.0.0-beta.47

$ npx create-electron-app electron-forge-app-webpack-d3 --template=webpack
$ cd electron-forge-app-webpack-d3
$ npm install && npm start
```
